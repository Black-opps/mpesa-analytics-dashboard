# MPesa Analytics Dashboard

A modern, production-ready analytics dashboard for MPesa transaction monitoring and analysis. Built with React, TypeScript, and Material-UI.

![Dashboard Preview](screenshot.png)

## ✨ Features

- **Real-time Analytics Dashboard** - Monitor MPesa transactions in real-time
- **Advanced Visualizations** - Interactive charts and graphs for transaction analysis
- **Customer Insights** - Track customer behavior and segmentation
- **Transaction Management** - Filter, search, and export transaction data
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile
- **Dark/Light Mode** - Toggle between themes for comfortable viewing
- **Multi-Cloud Ready** - Designed for deployment on AWS and OCI

Perfect for businesses, financial institutions, and developers who need to track and analyze MPesa payment flows with a modern, intuitive interface.

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- Git

### Installation

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/mpesa-analytics-dashboard.git
cd mpesa-analytics-dashboard
```
